# GPA Calculator

A modern, responsive GPA calculator web application.

## Features

- **Current Stats**: Input your current CGPA and completed credits.
- **Course Planner**: Add future courses with credits and expected grades.
- **Projected GPA**: Automatically calculates your potential new GPA.
- **Grading Scale**: Supports S (10), A (9), B (8), C (7), D (6), E (4) grading.

## How to Use

1. Enter your current GPA and completed credits (optional).
2. Add your courses for the upcoming semester.
3. Select expected grades for each course.
4. Click "Calculate Projected GPA".
